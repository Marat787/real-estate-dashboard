# 🏢 Разработка BI-системы для управления портфелем недвижимости

**Стек технологий:** Yandex DataLens, Google Sheets, Data Modeling, ETL-логика.

## 📋 Бизнес-контекст
Частный инвестор, имеющий объекты посуточной и долгосрочной аренды, нуждался в прозрачной системе контроля финансов и быстрого доступа к текущему статусу всех объектов.

## 🎯 Задачи
* Спроектировать и внедрить единое информационное пространство для учета всех бронирований.
* Автоматизировать расчеты финансовых метрик (остатки к оплате, чистая выручка, комиссии).
* Разработать интерактивный BI-дашборд для ежедневного операционного контроля и долгосрочной аналитики.

## ⚙️ Реализованное решение
* Внедрен процесс мануального ввода бронирований. На данном этапе отказ от API-интеграции.
* Разработана строгая структура таблиц, минимизирующая ошибки ручного ввода (настроены справочники статусов, объектов и источников).
* Внутри базы заложена автоматическая финансовая логика: формулы самостоятельно рассчитывают итоговую стоимость, комиссии и остатки к доплате.
* Настроена бесшовная интеграция подготовленных данных в **Yandex DataLens**.

## 📊 Визуализация (Yandex DataLens)

### 1. Операционный контроль (Вкладка «Объекты»)
Автоматический алерт-модуль для планирования работы клининга и встреч гостей, live-мониторинг занятости квартир и детализация по объектам с полной карточкой брони.

## 🏆 Результаты
* **Экономия времени:** Время на формирование ежедневной сводки сокращено (данные обновляются автоматически).
* **Управляемость:** Создан масштабируемый инструмент, позволяющий управлять объектами в режиме реального времени. Архитектура позволяет добавлять новые объекты без изменения логики системы.
